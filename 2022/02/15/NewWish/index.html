New Year, New Wish

發表於 2022-02-15 分類於 日誌

New year, new wish. For a better me.

其實今年已經過去兩個月了，但今天聽了Jserv教授的 Linux 課後心有所感，突然想寫一篇文章給明年的自己，希望之後我回來看到這篇文章的時候可以自信的說自己沒有浪費一年。

先從去年八月開始回顧吧。去年八月開了這個部落格，寫了一篇日誌作為起點，希望在未來記錄下自己的學習軌跡，至今也累積了十篇文章，其實超越了我自己的預估。但論文章品質來說，還有非常大的進步空間。我一直都是把事情做完才開始寫文章記錄，導致遺漏了很多容易被忽略的細節(儘管我當時也花很多時間在上面)。文章品質本是小事，實際上我只是想記錄自己的學習軌跡，不是當部落客，文章品質的部分慢慢改善就好。

這半年接了打工，寫了 web 專案後技能樹越來越往 web 偏，過程中花了很多時間，讓我漸漸的產生一些興趣，且欣賞 web 在軟體服務上的優秀，讓我不禁沉溺於此。於是這個寒假我花了很多時間寫校園選課心得的網站，希望建立一個新平台能更好的交流資訊(目前還在卡學校的 OAuth 單一簽入權限)。最初的規劃是希望能找更多人參與製作和維護的過程，但我在合作上的推廣並不特別的積極，最終幾乎是由我一人完成，還是讓我非常遺憾。就我的想法而言，希望能在校園和身邊燃起開源和合作的火種。

這三張是 wakatime 的時間統計，裡面只有算我寫 code 的時間，還沒有把測試和翻文章的時間算進去。總共加起來接近 90 小時了，除了年假以外，基本上都是不休息的努力。這段期間學到的東西不多，卻堆上一整個寒假，我也不得不捫心自問，這是否是我值得花上這麼多時間的事情。

唯一值得一提的是在寫打工地方的專案時，踩到了 Laravel 和 Inertia 的坑，搞懂問題成因後對兩個框架發了 PR 而且被成功採納，為我對開源社群的奉獻建立了起始的里程碑。這算是我這段期間最有收穫的一件事。

那麼今年，又要做些什麼改變和嘗試，讓自己變得更好呢?

至少在這個學期，我希望能達成跟上 Jserv 教授這學期開的 Linux 作業系統核心/實作的進度。每周都用最高標準檢視作業，並且完成。聽了教授今天的課程內容，我相信能完成這個目標實力必定會大有長進，希望我能堅持住。另一方面，NCU Plus 的計畫希望能持續推行，網站若能成功上線並且被廣泛使用，會是能推動開源社群前進的一大助力，也會是所有貢獻者的寶貴經驗。

這次的文章沒有太多校閱，基本上是要給我自己日後回顧用的，但我希望能把繼續向前的動力傳達給其他人。

謹筆。
